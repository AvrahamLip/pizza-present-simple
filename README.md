# Pizza Present Simple

A fun, educational web game designed to help learners practice the **Present Simple** tense through interactive pizza ordering!

## 🍕 Features
- **Interactive Gameplay**: Take pizza orders from a chef and build them correctly.
- **Educational Focus**: Practice present simple verb forms and vocabulary.
- **Responsive Design**: Play on desktop or mobile.
- **Modern Stack**: Built with React, TypeScript, and Vite.

## 🚀 Live Demo
Visit the game here: [https://avrahamlip.github.io/pizza-present-simple/](https://avrahamlip.github.io/pizza-present-simple/)

## 🛠️ Local Development

Follow these steps to run the project locally:

1. **Clone the repository**:
   ```sh
   git clone https://github.com/AvrahamLip/pizza-present-simple.git
   cd pizza-present-simple
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Start the development server**:
   ```sh
   npm run dev
   ```

## 📦 Tech Stack
- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
